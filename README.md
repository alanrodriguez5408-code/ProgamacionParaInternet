API de Biblioteca Inteligente
Este proyecto es una biblioteca digital construida con FastAPI y Python por Alan Rodriguez. 

## ¿Cómo usarlo?
1. Instala las herramientas necesarias abriendo la terminal y escribiendo: `pip install -r requirements.txt`
2. Enciende el servidor con: `uvicorn FASTAPPI_ALAN_rodriguez:app --reload`
3. Abre tu navegador web en la dirección: `http://127.0.0.1:8000/docs`
